# 海外商业化项目 — Cosmosebike eBike OS

**米娅** 的海外商业化 Agent 系统。距离 5 月展会不到 2 个月，**做减法、不造全自动工厂**：人扣扳机（决策、关系、点发送），AI 造子弹（查资料、写文案、做分析）。

---

## 2 个月冲刺原则：减法、人机协同

### 必须砍掉的「伪自动化」（立刻放弃）

| 砍掉 | 原因 | 人工替代 |
|------|------|----------|
| 全自动社媒发布与广告 API 调参 | Meta 原生算法已够用；API 调预算易跑飞，配置极繁琐 | **人工**在 Meta 后台建计划、调预算 |
| 官网 24/7 AI 销售客服 | €4,799 重决策商品，用户发现是机器人则信任归零 | 网页挂 **WhatsApp** 或试驾预约（如 Calendly），**人工**接高意向线索 |
| 全自动海量发信（B 端/PR） | 批量群发易进垃圾箱；欧洲车店和头部 KOL 拼的是走心不是走量 | **AI 写**「让他觉得你研究过他」的定制邮件，**人工**复制、用创始人邮箱发送 |

### 真正落地：带记忆的 Agent，人机协同

- **记忆库**：用 Gems / GPTs / Claude Projects 等给每个 Agent 固定知识库（品牌底线、历史表现、你的喜好）。不写代码、不串复杂 API。
- **流程**：**输入(人工)** → **处理(Agent)** → **执行(人工)**。AI 负责造子弹，人负责扣扳机。

## core 与 memory（全员必读）

- **core.md** = **所有精准信息、最新版本**。产品、定价、库存、试驾链接、品牌底线、最新 update，一律以 core 为准。对外文案、BD、话术前**必须参考 core**。
- **memory.md** = **对话的核心、产出的核心、想法的核心**。每次更新、每次对话、每次想法迭代后都可更新，让 AI 不断学习「我们真的想要什么」。不替代 core 的精准数据，而是沉淀我们的选择、偏好和声音。

由 **Mia 直接管理，Coach 辅助**。执行任何对外输出前：先读 **core**（精准数据），再结合 **memory**（我们要什么、怎么讲）。

## 三个真正落地的工作流（人机协同）

### 1. Hunter：爆款文案与数据复盘 Agent

- **记忆库**：品牌底线（锚点 Specialized/Trek、德国试驾、禁用「便宜」「性价比」）；历史表现（跑好的文案 A / 跑废的文案 B）；你的喜好（短平快、欧洲视角）。
- **落地流**：**输入(人工)** 本周要测的 3 张产品图 + 上周 Meta 导出数据 → **处理(Agent)** 分析数据（如「上周强调整车 9.9kg 转化最好」）、读图、输出 3 套优化文案（德/英） → **执行(人工)** 复制贴到 Meta 后台开跑。

### 2. Street：高管级 BD Agent（车店 / KOL）

- **记忆库**：合作模式（零压货寄售、试驾分成、代理等）；目标画像（高端公路车独立车店、非 Bosch 垄断；或 ~5 万粉硬核评测博主）。
- **落地流**：**输入(人工)** 扔一个 Google Maps 车店链接或 YouTube 博主链接 → **处理(Agent)** 抓取对方背景，生成一封「让他觉得你研究过他」的定制破冰邮件 → **执行(人工)** 检查后用自己的邮箱发送。一天 20 个精准 BD，转化率吊打群发 1000 封。

### 3. 展会与物料（Buzz + Street）

- 针对 5 月展会：德语易拉宝、Press Release、展台销售 FAQ 话术等。把中文大纲扔给 Agent，调取品牌记忆库 + 核心数据库，吐出符合欧洲习惯的展事物料。**B3 出文字，B4 Forge 出视觉**（易拉宝、桌布、传单、合作提案 PDF）；Street 参与发布与商业化视角。

---

## Dealer [D]：逼单话术

- **不追求面，只追求单点爆破**。对待每一个问询过的高意向线索，精准输出逼单话术。
- **提示词要最精细**：不需要懂投放，但必须极其精通「高客单价逼单」。
- **示例指令**：当客户问「为什么你们这么贵」时，**调用竞品对比话术**，并**立刻推送本地试驾预约链接**。所有话术与链接需从核心数据库取数（产品、定价、试驾地址）。

## 怎么用

1. **用 Cursor 打开本文件夹**，Agent 按本 OS 与技能运行。
2. **点名**：如「Hunter，按 A1 根据这张图出 Meta 文案」「Street，给这个车店写一封 BD 邮件」「Dealer，客户问为什么这么贵，给我一段逼单话术+试驾链接」「Forge，做 IG 方图：单手举车+9.9kg 大字」「Coach，核心数据更新了，帮大家同步」。
3. **写文案 / 话术前**：确保已读 **core.md** 与（若适用）**ads-performance.md** 本周数据。
4. **详细规格与提示词**：见 `.cursor/skills/cosmosebike-os/agents.md`。
