# Buzz 工作笔记：合作沟通前可准备的信息

**负责人**：**Buzz**。与 KOL/媒体沟通前，按本笔记提前搜索与准备，让对方感到**我们认真、准备充足**（非一次性群发）。

---

## 必读（最少要做）

| 信息 | 为什么 | 怎么查 |
|------|--------|--------|
| **对方最近 3–5 条/期内容** | 引用具体视频/文章标题或观点，证明看过 | YouTube 视频列表、网站最新稿 |
| **对方的一句话原则/立场** | 如「无审稿、可批评」「只与站得住的品牌合作」——邮件里引用一句，表示读过 | 专访（如 MTB-News）、About、某期口播 |
| **谁在讲 / 频道人设** | 是人名出镜还是团队、语言与地区 | About、视频里自我介绍 |
| **与我们产品的贴合点** | 一句：为什么 VAPOR SL 适合他/她（e-gravel、轻量、德语/英语区） | 对方 e-gravel/公路/电助力内容 |

---

## 加分（显得准备更足）

| 信息 | 为什么 |
|------|--------|
| **他们合作过的品牌** | 不对标竞品细节，只说明「我们知道您和 Canyon/FAZUA 等有过合作」，语境一致 |
| **他们的雷区或偏好** | 如 Felix 主频道不做 E-MTB、要无审稿——我们主动说「无审稿、可批评」 |
| **受众与地区** | 德语区 / 英国 / 国际——我们试驾、物流、话术可以对应（Mönchengladbach、德国现车） |
| **展会/线下** | 对方是否去 Eurobike 等——可提「若展会见面」「若来德可试驾」 |
| **联络方式与习惯** | 邮件 / 表单 / IG DM；是否注明 business 邮箱 |

---

## 我们这边一次备好（别让对方反复问）

| 材料 | 说明 |
|------|------|
| **PR 包** | 无附件正文 + 网盘高分辨率图、规格 PDF（agents.md B1） |
| **产品一句** | From 9.9kg、85Nm、250W rated、105 Di2、碳轮、Mönchengladbach 试驾（core 宣发口径） |
| **价格** | MSRP €5,199；特批买断 €4,000 仅内部/可说「专属合作价」不写成本价 |
| **试驾** | 德国 Mönchengladbach、Deruiz 合作、可预约 |

---

## 避免

- 只写「我们很期待合作」而不提对方任何具体内容。  
- 用错语言市场（对德语频道用英文套话、对英语区全德语）。  
- 写死未核实的数字（订阅量等以频道页为准，未核实则备注）。
