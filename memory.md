## memory 是什么

**memory = 对话里沉淀下来的选择与偏好**，不是精准数据（那是 core），也不是怎么配合、商业怎么想（那是 **.cursor/ethos.md**，企业组织核心价值观）。

- **core**：精准信息、数字、产品/定价/试驾/库存，写文案话术前必读。
- **ethos**：整体配合原则 + 商业落地怎么想，见 `.cursor/ethos.md`（宗旨 / 企业组织核心价值观）。
- **memory**：我们最近**决定**了什么、哪种过了哪种被否、哪句有效——随实践迭代，只留有用的。

## 对话 / 产出 / 想法核心（持续沉淀）

- **当前约束**：人可能过不去德国/现场，策略是**在线上实现最大的远程推动和曝光**。Hunter/Buzz/Street/Dealer 的产出与优先级都围绕「远程可执行、线上可放大」；不依赖本人在当地。

（例如：哪种文案风格通过/被否及原因；BD 邮件哪种开头转化好；Dealer 话术哪句最有效；品牌上我们决定强调什么、不强调什么。）


## 风险红线（我们的选择）

| 风险 | 我们的处理 |
|------|------------|
| 无 CE 认证 | 不主动提认证，文案不出现 "road legal" "certified" 等词 |
| 600W 在欧盟 | 可写 600W peak performance；建议同时写 250W rated 或 85Nm，避免误读 |
| Coach 信息核实 | 对外/KOL/渠道数据须 cross-check、有来源；不确定就备注「来源未核实/可能不准」 |
| Bafang=DIY 印象 | 对外用 "precision mid-drive system"，不强调 Bafang 名字 |
| 零品牌知名度 | 所有物料突出 Deruiz 合作 + 实体地址，建立本地信任 |
| 续航夸大 | 始终用「诚实版」数据 + 条件注释，主打 120km |
